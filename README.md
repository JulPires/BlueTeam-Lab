#  BlueTeam-Lab

Welcome to my Blue Team laboratory.

This repository documents my hands-on journey learning defensive cybersecurity through practical labs, investigations and incident response exercises.

The goal is not only to learn tools, but to understand how to investigate, detect and respond to security incidents while documenting every step.

---

#  Objectives

- Learn Blue Team fundamentals.
- Practice network traffic analysis.
- Investigate Windows logs.
- Analyze Indicators of Compromise (IOCs).
- Develop Incident Response skills.
- Build a professional cybersecurity portfolio.
